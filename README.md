# Run Expectancy App

A run expectancy visualizer that covers:

- Base state
- Game state
    - Current inning
    - No. outs
- Ball/strike count

This is a Cordova project build using the Ionic framework.

Run expectancy data was calculated using Retrosheet.
Data is an aggregate of 2010-2013.
